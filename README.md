MindList
Descrição

A MindList é uma aplicação móvel desenvolvida em Flutter com o objetivo de auxiliar os utilizadores na gestão de tarefas e organização pessoal.

A aplicação integra um calendário interativo com um sistema de gestão de tarefas, permitindo organizar compromissos, visualizar atividades diárias e acompanhar tarefas de forma simples e intuitiva.

Este projeto foi desenvolvido no âmbito da Prova de Aptidão Profissional (PAP) do Curso Profissional de Técnico de Gestão e Programação de Sistemas Informáticos.

Funcionalidades
Calendário interativo;
Criação e gestão de tarefas;
Visualização de tarefas por data;
Gestão de tarefas em atraso;
Tema escuro;
Configuração do primeiro dia da semana;
Exportação de dados em formato JSON;
Interface simples e intuitiva.
Tecnologias Utilizadas
Flutter
Dart
